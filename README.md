# Path Planning and Navigation in Robotics

Educational platform demonstrating path planning algorithms with interactive multiple algorithms visualization.

## Interactive Demo Usage

1. Click "Set Start" and place green start point
2. Click "Set Goal" and place red goal point
3. Click "Toggle Obstacles" and draw black walls
4. Click "Find Path" to watch A* algorithm work
5. Experiment with different maze configurations
